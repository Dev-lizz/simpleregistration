--Português --
O presente projeto é um formulário simples que desenvolvi com conhecimentos adquiridos no curso técnico 
em informática que realizei há alguns anos. Estou retornando à área de T.I. e decidi começar com um projeto simples,
focado apenas no básico de HTML, CSS e JavaScript, com o auxílio de inteligência artificial e da documentação para
relembrar alguns conceitos. O projeto não possui banco de dados nem qualquer tipo de validação, pois o foco
principal é a parte visual do front-end.

--English--
This project is a simple form I developed using knowledge acquired during a technical IT
course I completed a few years ago. I am returning to the IT field and decided to start with a
simple project, focusing only on the basics of HTML, CSS, and JavaScript, while using artificial
intelligence and documentation to refresh some concepts. It does not have a database or any
type of validation, as the main focus is the visual part of the front-end.
